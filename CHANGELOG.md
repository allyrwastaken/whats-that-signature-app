# Changelog

The notes under each version are shown in the in-app "Update available" window.

## v1.0.0
- The app window stays open after saving (with a confirmation), so you can keep adjusting settings.
- Cleaner overlay text: removed the drop shadow, kept the outline.
- Much more accurate identification — each mineral and category now respects its real maximum rock count, ruling out impossible guesses.
- Round-number signatures that could be more than one thing now list every option (e.g. "Salvage / FPS ×8 / ROC ×6").
- Updated names: Salvage/Harvestables, FPS Mineables, and ROC Mineables; minerals use Ice / Silicon / Aluminium.
- ROC Mineables are now recognised all the way up to 80,000 (20 rocks).
- Salvage is now "Salvage/Harvestables" (it also covers debris) and shows just the name — no rock count.
- Wider signature ranges: Salvage/Harvestables up to 30,000 and FPS Mineables up to 60,000.
- More reliable readings over bright yellow moon surfaces, where white signature text used to blend into the background.
- Each release now ships with a SHA-256 checksum and a verifiable build-provenance attestation.
